# Amal William Portfolio

This is my Personal portfolio website, built to showcase professional experience, technical skills, education, projects, contact details, and CV access.

## Overview

This portfolio presents my profile as a Full Stack Developer with experience in PHP, ColdFusion, Python, SaaS development, enterprise applications, legacy modernization, and responsive web development.

## Features

- Responsive portfolio layout for desktop, tablet, and mobile screens
- Sticky navigation header
- CV view button in the header
- About, skills, education, work experience, projects, and contact sections
- Grey and white visual theme
- Project cards with GitHub links
- Contact form UI

## Tech Stack

- HTML5
- CSS3
- JavaScript
- Responsive design with CSS media queries

## Work Experience Highlight

- Junior Software Engineer at Loremine Technologies
- Software Technical Consultant at Loremine Technologies
- Project Trainee at Spiderline Technologies

## How To Run Locally

Clone the repository:

```bash
git clone https://github.com/amalWILLIAM/PORTFOLIO.git
```

Open the project folder:

```bash
cd PORTFOLIO
```

Open `index.html` in your browser.
